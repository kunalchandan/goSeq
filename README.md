# GoSeq
Alignment for de-novo sequences.

See more at my blog post [linked here.](https://chandan.one/posts/GoLang-Refresher/)

# Running
```bash
go get .
go run .
```

# Testing
Test all
```bash
go test .
```

Test individual
```bash
go test -timeout 30s -run ^TestDrawMatricies_MediumMismatched1$ goSeq
```
